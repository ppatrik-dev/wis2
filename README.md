A **modular web-based information system** built with **Laravel (PHP 8.1)** that provides comprehensive management of users, courses, scheduling, and academic activities. The system implements **role-based access control**, a structured modular architecture, and a modern responsive UI built with **Flowbite + Tailwind CSS**.
